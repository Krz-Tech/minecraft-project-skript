# krz-gui - カスタムGUIメニューシステム

統合メニューシステムです。主要機能へのクイックアクセスを提供します。

## 機能

- 統合メインメニュー
- プロフィール表示
- 経済メニュー
- UXガイドライン準拠のデザイン

## コマンド

### `/menu`
**説明**: メインメニューを開きます。

```
/menu
```

**または**: 棒を右クリック

## メインメニュー構成

| 項目 | 機能 | 状態 |
|------|------|------|
| プロフィール | ステータス・ランク表示 | ✅ 実装済み |
| 経済 | 残高・取引履歴 | ✅ 実装済み |
| 装備管理 | 装備変更画面 | 🚧 準備中 |
| プレイグラウンド | 出撃・戦績 | 🚧 準備中 |
| 拠点 | テレポート・管理 | 🚧 準備中 |
| 設定 | 各種設定 | 🚧 準備中 |

## プロフィールメニュー

表示内容:
- プレイヤー名
- ランク（開発中）
- HP, DEF, ATK, 会心率, 会心ダメージ
- 残高

## 経済メニュー

表示内容:
- 現在の残高
- 取引履歴（開発中）
- 建材ショップ（開発中）

## デザイン仕様

UXガイドライン (`GameSystem/3.GameSystem_UX.md`) に準拠:
- **背景**: 灰色ガラス板
- **カラー**: Cyan/Gold/Grayパレット
- **サウンド**: GUI開閉、ボタンクリック

## 技術メモ

- SkBeeの仮想インベントリを使用
- 各メニュー間の遷移は1tick待機後に実行
- アイコンとLoreはUXガイドラインに準拠
